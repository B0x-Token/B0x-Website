This is a full front end for B Zero X or B0x Token.

If you want to run this folder locally there are a few options.

Windows
1) Install python then run `cmd`
2) Change Directories in cmd to be inside the website folder.
3) type `python -m http.server` then visit localhost:8000 on ur web browser to view the site
4) Follow the link it generates to view your self hosted dAPP front end

Ubuntu
1) run a terminal in the website folder and type `http-server`
2) Follow the link it generates to view your self hosted dAPP front end
